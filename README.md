# KalshiAlpha

A trading terminal application built with React, Vite, and Electron.

## Project Setup

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later recommended)
- npm (included with Node.js)

### Installation

```bash
npm install
```

### Development

Start the Vite dev server (browser mode):

```bash
npm run dev
```

Start the Electron dev server:

```bash
npm run electron:dev
```

### Build

Build for web:

```bash
npm run build
```

Build the Electron desktop app:

```bash
npm run electron:build
```

### Preview

Preview the production build locally:

```bash
npm run preview
```

Preview as an Electron app:

```bash
npm run electron:preview
```
